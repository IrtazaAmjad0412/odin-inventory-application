# odin-inventory-application

A simple web application built with **Node.js**, **Express**, **EJS**, and **PostgreSQL** to manage categories and their associated items. This app allows users to create, read, update, and delete categories and items.

---

## Features

- List all categories
- Add, edit, and delete categories
- View items under a specific category
- Add, edit, and delete items
- Organized routing with Express routers
- Uses PostgreSQL for data storage

---

## Tech Stack

- Node.js
- Express.js
- EJS (Embedded JavaScript templates)
- PostgreSQL
- dotenv for environment variables

---

## Installation

1. **Clone the repository**

```bash
git clone <your-repo-url>
cd <repo-folder>
```

2. **Install dependencies**

```bash
npm install
```

3. **Start the server**

```bash
node app.js
```
